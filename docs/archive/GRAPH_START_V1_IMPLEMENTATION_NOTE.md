# Graph-Start v1 Skeleton — Implementation Note

## Что изменено
- Старт пересобран в сцену графа с выраженным ядром, пространственной топологией доменов и реактивными связями.
- 5 major domains представлены как узлы-территории вокруг центра, а не список строк.
- При выборе домена граф меняет фокус: ядро усиливается, связанные домены поднимаются, дальние домены приглушаются.
- Первый слой глубины и встроенные инструменты появляются как структурное расширение выбранного домена внутри граф-сцены.

## Что реально работает сейчас
- Выбор стартового домена как точки входа в живую карту.
- Видимые междоменные рёбра с режимами muted/visible/focused.
- Semi-dynamic поведение: пульс ядра, мягкое дыхание узлов, усиление центра и рёбер по выбору.
- Узлы глубины и tool entry points отображаются в привязанной зоне расширения выбранного домена.
- Структурный next-step блок с честной формулировкой (без фейковых прогнозов).

## Что сознательно упрощено
- Deep route engine пока не реализован: показывается структурный следующий ход, а не полноценный вероятностный маршрут.
- Инструменты пока работают как входные точки и сигналы готовности, без полной глубокой логики каждого инструмента.
- Unlock-условия локальные и детерминированные, ещё не связаны с полным циклом Data Substation.
